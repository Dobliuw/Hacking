# Resistance

La **resistencia eléctrica** es la oposición que un material presenta al paso de la corriente leéctrica por su estructura atómica (Flujo de electrones) y como vimos en [[2. Key Concepts#Resistance]] se mide en *ohmios* (*Ω*).

----
# Resistor

El componente conocido como **resistencia** (*Resistor*) es un elemento pasivo fundamental en electrónica que se opone al flujo de corriente, limitándlo para proteger otros componentes, dividir voltajes o ajustar el comportamiento de un circuito. 

![[Pasted image 20251216215715.png|600]]

> [!warning] Problems fixed by Resistors
> En electrónica casi nunca querremos corriente "ilimitada", sin resistencia:
> - Un LED se quema.
> - Un pin GPIO se destruye.
> - Una batería se descarga violentamente.
> - Un chip muere.
>   
> Es por estas y otras carácteristicas que las **resistencias nos permiten**:
> - Limitar corriente.
> - Crear caídas de tensión controladas.
> - Definir estados lógicos.
> - Estabilizar señales
> - Proteger componentes.

# Resistors types

Las resistencias se clasifican primero por *cómo se montan en la placa*, no por su valor:

###### Through-Hole (THT / PTH)

Estas resistencias se sueldan pasando los terminales por un agujero de la placa, por lo general suelen ser más grandes, más fáciles de manipular así como más tolerantes al calor. Estos tipos de resistencias son ideales para prototipos, aprendizaje y alta potencia.

![[Pasted image 20251216215743.png|400]]

Se suelen utilizar en:

- Breadboard.
- Prototipos.
- Electrónica educativa.
- Potencia media/alta.
- Reparaciones.
###### Surface Mount Device (SMD)

Estas son las "resistencias sin patitas", se sueldan sobre pads superficiales ya que son mu pequeñas. Estas resistencias tienen menor inductancia y capacitancia parásitra y son ideales para altas frecuencias, automatizables (Pick & Place) y tinen menorp otencia individual.

![[Pasted image 20251216220038.png|500]]

Se suelen utilizar en:

- PCBs modernos.
- RF.
- IoT.
- ESP32.
- Electrónica comercial

# Resistors code 

Los **códigos de color** (`Color Code`) en las resistencias es un *sistema de bandas coloreadas* en una resistencia que indican mediante reglas su *valor eléctrico* (`Ohmios`), *tolerancia* (Que tan preciso es el valor) y coeficientes de temperatura.

##### Through-Hole Resistors - COLOR CODE

Una resistencia codificada por colores *siempre se interpretará* de la siguiente manera:

> [!info] $$ figuras * multiplicador = resistencia (Ω) +/-  tolerancia $$

Todo código de colores seguirá dicha estructura.

![[Pasted image 20251216220126.png]]

##### Sourface Mounted Device (SMD) - NUMBER CODE

A diferencia de las *THT*, las **SMD no usna colores**, usán *códigos númericos impresos*. Estos digitos también pretenden indicar el valor eléctrico de la resistencia (`Ohmios`).

![[Pasted image 20251216220256.png]]

Una resistencia SMD codificada por números *se interpretará* de la siguiente manera:

> [!info]  $$WX/Y/Z = WX/Y/ * 10^Z$$

Ejemplos:

- 1001 => **1KΩ** (*100* * *10^1*).
- 4702 => **47KΩ** (*470* * *10^2* ).
- 3300 => **330Ω** (*330* * *10^0*).

##### Sourface Mounted Device (SMD) - E96 SYSTEM

El **sistema de códigos E96** se refiere principalmente a dos cosas en eletrónica: un sistema de numeración preferente para resistencias de *alta presión* (Tolerancia del 1%) que usa un marcado especial de tres caracteres (Dos dígitos + una letra) para identificar valores exactos.
