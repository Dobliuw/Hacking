# Soldering Station + Hot Air


Una **estación de soldadura** regulable en temperatura permite fundir estaño para unir componentes con pads, pines o pistas. En electronica o hardware hacking puede usarse para modificar PCBs, soldar puertos UART, reparar pistas, montar sensores y headers en sistemas embebidos o simplemente crear.

Cuanto mejor el control térmico, menos riesgo de delaminar la PCB o quemar integrados sensibles.

![[Pasted image 20251210104517.png|600]]

Sirve para desoldar o soldar componentes SMD, QFN, BGA pequeños, memorias Flash, reguladores, etc.

# Tin (Solder Wire)

Aleación de estaño + plomo (60/40 o 63/37) o libre de plomo. Se utiliza para unir componentes electrónicos y eléctricos mediante calor, creando una conexión sólida y conductora; es blandoy, se funde a temperaturas esepecíficas y contiene *fundente* (**Flux**) interno para limpiar las superficies y facilitar la unión, evitando la oxidación y asegurando una buena conexión eléctrica.

![[Pasted image 20251210104638.png|600]]

# Desoldering Tool

Un **desoldador** es una bomba de vacío manual utilizada para retirar estaño fundido. Se usa para *liberar pines*, *retirar headers*, *reparar errores de soldado* o preparar pads antes de capturar señales. 

![[Pasted image 20251210104714.png|600]]

# Desoldering Wire

El **cable para desoldar** es la herramienta fina para limpiar estaño residual. Ideal para:

- Dejar pads expuestos limpios.
- Eliminar cortos microscópicos en SMD.
- Preparara footprint para micrscopía o testeo.
Indispensables cuando trabajmos on PCBs densas tipo routers, IoT o placas de teléfonos.

![[Pasted image 20251210104842.png|600]]

# Cutting Pliers and Nippers

**Pinzas de corte de precisión**. Sirven para recortar pines, cbales, patas de componentes o limpiar PCBs. En hardware hacking se usan para cortar líneas de alimentación o señales cuando necesitamos aislar un rail para medir consumo, glitching o manipulación de buses.

![[Pasted image 20251210105213.png|600]]
# Screwdrivers

Un **Set de destornilladores de precisión** utilizados para abrir dispositivos.

![[Pasted image 20251210105412.png|600]]

# Multimeter

Un **multimetro** es un instrumento fundamental para *medir voltajes*, *continuidad*, *resistencia* y *corrientes*. En general se utiliza con el fin de análizar y medir voltaje en un circuito electrónico, pero en hardware hacking puede ser utilizado para:

- Identificar GND, VCC y señales en puertos expuestos.
- Verificar si un bus es de *3.3V*, *1.8V* o *5V* antes de conectarse.
- Mediro consumo (Útil para side-channler)

![[Pasted image 20251210105919.png|600]]

# Oscilloscope

Un **Osciloscopio** nos permitirá ver señales en el tiempo: forma de onda, voltaje, ruido, edges, frecuencias. En hardware hacking es central para:

- Identificar si un pin es *UART*, *SPI*, *I2C* o simplemente alimentación.
- Analizar relojes.
- Depurar transmisiones digitales.
- Medir glitches o fallas temporales.
- Ver activity patterns de chips cuando querés inferir comportamiento.

![[Pasted image 20251210105949.png|600]]

# PCBite

La **PCBite** es un sistema de soportes magnéticos para sujetar PCBs y puntas de osciloscopio/lógica. Evita manos temblorosas y es muy útil cuando:

- Capturamos señales en pads microscópicos.
- Soldamos en ángulos incómodos.
- Necesitamos estabilidad para glitching, reading o sniffing.

![[Pasted image 20251210110137.png|600]]

# Logic Analyzer

Un **analizador lógico** es un instrumento de medida que captura los datos de un circuito digital y los muestra para su posterior análisis. Es la herramienta digital indispensable ya que captura buses como:

- UART.
- SPI.
- I2C.
- SWD.
- JTAG (Parcialmente).
- 1-Wire

Y los decodifica, permitiendonos ver *el protocolo crudo*.

![[Pasted image 20251210110344.png|600]]

# Digital Microscope

Un **microscopio digital** e fundamental para realizar la inspección de pistas, micro soldaduras, análisis de daños, inspección de pads, lectura de números de chips, etc.

![[Pasted image 20251210114826.png|600]]

# Precision Tweezers (Anti-Static)

Las **pinzas de precisión anti estática** son clave para colocar componentes SMD, manipular jumpers, sujetar cables de prueba, etc.

![[Pasted image 20251210114915.png|600]]

# Adjustable Power Supply (Bench PSU)

La **fuente de alimentación regulable** nos permite controlar voltaje y corriente, actividad crítica cuand oalimentamos placas desnudas, esto nos permitirá:

- Evitar quemad dispositivos.
- Detectar cortos.
- Medir consumo con precisión.
- Hacer brown-out attacks o glitching básico.
