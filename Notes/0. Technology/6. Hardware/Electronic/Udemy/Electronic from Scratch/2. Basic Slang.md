# Voltage

El **voltaje** es la diferencia de energía que una carga eléctrica (Como un electrón) experimenta entre dos puntos de un circuito debido a un campo eléctrico; es una *medida de cuánta energía está disponible para mover esa carga de un punto a otro*. Esto indica cuánta energía por unidad de carga está disponible para mover electrones a través de un material. Un voltaje mayor indica un campo eléctrico más intenso y, por lo tanto, una mayor capacidad de producir corriente si el circuito lo permite.

El voltaje...

- *NO* es corriente.
- *NO* es cantidad de electrones.
- *NO* es presión.

Se mide en `voltios` (`V`).
