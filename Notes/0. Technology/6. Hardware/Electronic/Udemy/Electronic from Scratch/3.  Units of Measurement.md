# Metric System

El **sistema métrico** es un sistema de unidad de medida *universalmente usado* basados en la siguiente tabla:

| Quantity    | SI Unit  | Unit Abbreviation |
| ----------- | -------- | :---------------: |
| Time        | *Second* |        `s`        |
| Length      | *Meter*  |        `m`        |
| Mass        | *Gram*   |        `g`        |
| Temperature | *Kelvin* |        `K`        |
| Force       | *Newton* |        `N`        |
Donde las unidadses se relacionan por potencias de 10 (Múltiplos y submúltiplos), haciendo las converciones fáciles como 100cm = 1m.

# Metric Prefixes

##### Well Know

| Prefix (Symbol) | Power | Numeric Representation |
| --------------- | ----- | ---------------------- |
| *kilo* (`k`)    | 10^3  | 1.000                  |
| *hecto* (`h`)   | 10^2  | 100                    |
| *deka* (`da`)   | 10^1  | 10                     |
| -               | 10^0  | 1                      |
| *deci* (`d`)    | 10^-1 | 0.1                    |
| *centi* (`c`)   | 10^-2 | 0.01                   |
| *mili* (`m`)    | 10^-3 | 0.001                  |
Ejemplos:

- 1 `c`m = 0.01m
- 1 `k`g  = 1000g
