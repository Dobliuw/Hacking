# Learning 

Cuando hablamos de **aprendizaje de modelos de ML**, realidad estamos describiendo *qué tipo de información recibe el modelo mientras aprende*.

La diferencia central entre los paradigmas es *qué tipo de feedback existe durante el entrenamiento*.

# Supervised Learning

Este es el paradigma más común en machine learning práctico. 

Aca el modelo aprende usando *datos etiqeutados*. Cada ejemplo del dataset ya tiene la respuesta correcta asociada. 

El modelo ve muchos ejemplos y empieza a aprender *patrones estadísticos que separan las clases*.

Con suficiente ejemplos, los sistemas pueden aprender que, por ejemplo, palabras como:

- Free.
- Crypto.
- Win.
- Urgent.

Auemtan la posibilidad de spam. Lo importante es que durante el entrenamiento *el modelo sabe si la predicción fue correcta o incorrecta*, y ajust sus parámetros para mejorar.

# Unsupervised Learning

En el **aprendizaje no supervisado**, el dataset *no tiene etiquetas*. El modelo solo recibe datos sin ninguna indicación de qué significan.

Podrían ser logs de red, features de tráfico, comportamientos de usuario, etc.

Pero el modelo no posee información de qué es un comportamiento malicioso y que es normal.

El objetivo entonces cambia completamente: En lugar de predecir etqieutas, el modelo intente *descubrir estructura oculta en los datos*, por ejemplo agrupando las conexiones tícpicas en un lado y encontrando comportamientos raros que terminarían siendo maliciosos. (El agrupar datos similares se conoce como **clustering**).
