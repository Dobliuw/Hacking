# TGPT - LLM into the terminal

**TGPT** es un cliente no oficcial para usar modelos tipo ChatGPT desde la terminal (CLI). Esta herramienta actúia como un wrapper que envía promps a APIs de modelos simulando el comportamiento de un cliente web pero en terminal. 

Esta herramienta puede ser una gran opción para casos de uso tales como:

- Testing rápido desde CLI sin abrir navegador.
- Automatizacción simple (Script en bash, python, pipelines, etc.).
- Integración en entornos minimalistas.

> [!info] Considerations
> Es importante tener en cuenta que esta herramienta nos permite conectar con modelos de AI mediante el uso de APIs, por lo que se vuelve una opición interesante a la hora de querer combinar este tipo de herramientas y funcionamientos con los ya vistos y descritos, como [[0. Running Local AI]], para aumenta la privacidad de nuestros datos.

- Instalación.
```bash
curl -sSL https://raw.githubusercontent.com/aandrew-me/tgpt/main/install | bash -s /usr/local/bin
```

- Correr de manera local.
```bash
tgpt --preprompt "$preprompt" --provider ollama --url http://localhost:1236/v1/chat/completions --model "llama-3.2-3b-instruct" "Hola como estas?"
```

Podríamos instalar **slow** para parsear los resultados y verlos de una manera mucho más linda en consola tratandose de *.md*. Mismo podríamos tener un script en `.bashrc`:

```bash
function wgpt(){

    read -r preprompt << 'EOF'
    .... SYSTEM PROMPT
EOF
     prompt="$@"
 
     /usr/local/bin/tgpt --preprompt "$preprompt" --provider ollama --url http://localhost:1236/v1/chat/completions --model "llama-3.2-3b-instruct" "$prompt" | tee -a /tmp/ai
 
     status=${PIPESTATUS[0]}
 
     if [ $status -eq 0 ]; then
         /usr/bin/glow /tmp/ai && rm -rf /tmp/ai
     else
         /opt/localAI/lmStudio &>/dev/null & disown 
         sleep 10
         wgpt "$prompt"
     fi
 }

```

Esta función se encargará de correr el modelo llama 3 si no esta cargado y parsar lo que ejecutemos tal como: 

```bash
wgpt "Podrías darme un script de Python con Scappy que se encarge de escanear una red con el fin de pasar un parametro tal como --network 192.168.1.30/24"
```

# Warp

**Warp** es una terminal moderna (Tipo reemplazo de bash/zsh UI) potenciada con AI. Esta terminal corre una shell real pero encima agrega:

- UI estructurada (Bloques de comandos en lugar de texto plano).
- Motor de autocompletado inteligente.
- Integración con LLMs para sugerenias y generación de comandos.

Esto es importante porque nos permite parsear comandos cocmo objetos, dandonos lugar a mayor automatizaión que nos facilitará el debugging y la reproducibilidad.

Para *instalar* Warp nos dirigimos a [su web](https://www.warp.dev/download) y seleccionamos el paquete que corresponda.

# Gemini CLI

**Gemini CLI** es la CLI oficiall de Googlle para interactuar con modelos Gemini desde terminal. Esta herramienta nos permitirá realizar prompts simples, ejecución de tareas complejas e integración con scripts, siendo el cliente oficial haiendo uso de APIs de Google (Gemini API / Vertex AI) y autentación vía API Key.

- [Instalación](https://geminicli.com/):
```bash
npm install -g @google/gemini-cli
```
