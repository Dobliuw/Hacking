# Setting up 

Para dar comienzo a una campaña de phishing vamos a necesitar información sobre nuestro objetivo. En este punto bien podríamos tener la información brindada por el cliente o recurrir a técnicas de [[OSINT - Open Source Intelligence]] para obtener:

1. Direcciones de correo electrónico de los empleados objetivo.
2. Información general útil (*Lenguaje*, *Zona horaria*, *Intereses* (Esto dependiendo del Scope definido [[1. Incrementing Success on Phishing#Defining Type and Scope]])).

Una vez que se tenga esta información, necesitaremos nuestro propio **servidor SMTP** (**Simple Mail Transfer Protocol**).
