# Phishing

El **Phishing** es una técnica de *ingenería social* cuyo objetivo es e*ngañar a personas para que revele información sensible* (Credenciales, datos financieros, tokens), *realicen una acción que comprometa seguridad* (E.g. transferir dinero, ingresar credenciales) o instalen software malicioso. El vector más común es el correo electrónico, pero también se pueden ver con *SMS* (*Smishing*), *Llamadas* (*Vishing*), mensajería instantánea y páginas web falsas.

---
# Phishing Types

Los tipos de Phishing que se pueden llevar a cabo o encontrar en campañas masivas son:

- **Spear-phishing**: Phishing dirigido, personalizado a una víctima o grupo reducido.
- **Whalling**: *Spear-phishing* dirigido a altos ejecutivos.
- **Clone phishing**: Se duplica un correo legítimo y se reemplazan enlaces/adjuntos maliciosos.
- **Smishing** / **Vishing**: Phishing mediante SMS o voz (Llamadas telefónicas), respectivamente.
- **Business Email Compromise** (**BEC**): Fraude que simula órdenes o solicitudes empresariales (Normalmente sin malware).

----
# Phishing Campaing

Una **campaña de Phishing** es un conjunto planificado de avtividades que usan phishing como táctica para alcanzar un objetivo específico (Robar credenciales de empleados, obtener acceso a una red corporativa, fraude financiero, etc.). Una campaña incluye fases (Reconocimienot), preparación de señuelos, envío/entrega, explotación, y evaluación; y habitualmente está dirigida a un grupo de víctimas (Masivo) o a individuos concretos (*Spear-phishing*).

- **Target**: Definir objetivo estratégico (Credenciales, datos sensibles, acceso inicial).
- **Recon**: Recolección de información pública ([[OSINT - Open Source Intelligence]]) sobre la organización/individuos (Roles, direcciones de correo, proveedores, tecnologías, etc.).
- **Decoy Design**: Mensaje, web y/o artefactos que aumentan credibilidad (Lenguaje, remitentes, aparantes, marcas).
- **Channel** / **Vector**: Correo electrónico, mensajería instantea, voz, web, SMS o convinaciones especiales de estas.
- **Distribution** / **Interaction**: La víctima realiza la acción esperada (Ingresa credenciales, descarga adjunto, accede a una web, hace click, etc.).
- **Metrics**: Medición de éxito y lecciones parnedidas (Tasas de clicks, credenciales obtenidas, detecciones).

# Artifacts to increase the Success

El ser humano es vulnerable por naturaleza a los mismos estimulos, a continuación algunos de los elementos que nos pueden brindar un aumento considerable en la capacidad de éxito de campañas de este tipo:

- **Personalization** (Adjuntar nombre, puesto, contexto).
- **Urgency** and **Time pressure** (Plazos, avisos de seguridad, necesidad).
- **Fears** (Situaciones temidas, urgencia vital).
- **Apparent Authority** (Remitente convincente, logos, branch).
- **Credible Context** (Documentos/URLs relacionados con proveedores conocidos).

# Impact

- Compromiso de cuentas y credenciales.
- Acceso lateral y exfiltración de datos.
- Fraude financiero (Ordenes de pago fraudulentas).
- Pérdida de reputación y sanciones regulatorias.
- Coste operativo por respuesta al incidente.
